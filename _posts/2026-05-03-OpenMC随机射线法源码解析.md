## 1. 从 MC 到 RR：思维转换

### 1.1 MC 在做什么

MC 解的是输运方程的**积分形式**：模拟一个个中子的生命史。每个中子从源出发，沿某个方向飞行，碰或不碰由截面概率抽样决定。一次 batch 内，中子经历的所有碰撞、散射、裂变都被完整追踪——散射源是**隐式且完全收敛**的。

### 1.2 确定论方法在做什么

确定论方法解的是输运方程的**微分形式**。核心思路是：把方程降维成沿一条直线的常微分方程（ODE），然后**解析求解**。

最关键的一步是**特征线法（Method of Characteristics, MoC）**：沿一条特征线 $\mathbf{r}(s) = \mathbf{r}_0 + s\boldsymbol{\Omega}$（其中 $\mathbf{r}_0$ 为射线起点位置，$\boldsymbol{\Omega}$ 为射线方向单位向量，$s$ 为沿射线的弧长 cm），输运方程化为：

$$\frac{d\psi}{ds} + \Sigma_t \psi(s) = Q(s)$$

其中 $\psi(s)$ 为沿射线的角通量（angular flux，单位 n·cm⁻²·s⁻¹·sr⁻¹），$\Sigma_t$ 为宏观总截面（cm⁻¹），$Q(s)$ 为源项（含散射和裂变贡献，n·cm⁻³·s⁻¹·sr⁻¹）。

> **ODE 的求解框架**：
> - **已知量**：$\Sigma_t$（材料截面，由输入给定）、$Q$（源项，由上轮迭代结果计算）、射线穿行长度 $\ell$（由几何追踪得到）
> - **未知量**：$\psi(s)$——角通量沿射线的分布，是我们要求解的目标
> - **求解方法**：这是一个标准的一阶线性 ODE，在平源假设下可以直接写出解析解

这是一个一阶线性 ODE，在平源假设下（$Q$ 恒定）有**解析解**：

$$\psi^{out} = \psi^{in} \cdot e^{-\Sigma_t \ell} + \frac{Q}{\Sigma_t}(1 - e^{-\Sigma_t \ell})$$

其中 $\psi^{out}$ 为射线穿出该 FSR 时的出射角通量，$\psi^{in}$ 为射入时的入射角通量，$\ell$ 为射线在该 FSR 中的穿行长度（cm），$e^{-\Sigma_t \ell}$ 为指数衰减因子（表示未被碰撞的透射份额）。右端第二项中 $Q/\Sigma_t$ 的物理含义是无限大均匀介质中的平衡角通量——即源产生与碰撞吸收达到平衡时的角通量值。

> **解析解的含义**：
> - **已知量**：$\psi^{in}$（入射角通量）、$Q/\Sigma_t$（该 FSR 的平衡角通量）、$\Sigma_t \cdot \ell$（光学厚度）
> - **输出**：$\psi^{out}$（出射角通量）= 透射部分 + 源填充部分
> - **物理图像**：右端第一项是入射通量经过衰减后的残余（$\psi^{in}$ 未被碰撞的份额），第二项是 FSR 内部源项对出射通量的贡献。当 $\ell \to \infty$ 时第一项趋于 0，$\psi^{out} \to Q/\Sigma_t$（完全被源填充到平衡态）

MC 需要大量粒子历史来逼近真解，而 MoC 只需要沿足够多的特征线调用这个解析公式就能求解。

### 1.3 随机射线法 = 随机化的 MoC

传统 MoC 确定性地选取特征线（均匀角度求积 + 均匀空间间距），需要存储所有特征线端点的角通量用于迭代。RR 的改动很简单：

| | 传统 MoC | 随机射线法 |
|---|---------|----------|
| 射线选取 | 确定性求积 | **每轮迭代随机采样** |
| 射线间角通量 | 存储、复用 | **不存储，用完即弃** |
| 收敛方式 | 固定点迭代 | **inactive/active batches（类似 MC）** |
| 并行策略 | 区域分解 | **射线级并行（类似 MC）** |

这个改动看似微小，但带来了巨大好处：不需要存储角通量（省内存）、可以自然处理复杂 CSG 几何、全域方差近似均匀。

### 1.4 关键术语对照

| MC 术语 | RR 术语 | 含义 |
|---------|---------|------|
| particle（中子） | ray（射线/特征线） | RR 的射线不代表中子 |
| batch | batch（幂迭代） | 概念相同，但含义不同 |
| inactive batch | inactive batch | MC 等裂变源收敛，RR 等散射+裂变源都收敛 |
| particle count | ray count | 通常远小于 MC 的粒子数 |
| collision | 不存在 | RR 用解析公式，不模拟碰撞 |
| weight | 无权重变化 | 每条射线权重相同 |
| k_eff 更新 | 相同公式 | $k^n = k^{n-1} \times F^n/F^{n-1}$ |

> 其中 $k^n$ 为第 $n$ 轮迭代的有效增殖因数，$F^n = \sum_i V_i \cdot \nu\Sigma_f \cdot \phi^n$ 为全局总裂变率（$V_i$ 为 FSR 体积，$\nu\Sigma_f$ 为中子产生截面，$\phi^n$ 为标量通量）。
>
> **k_eff 更新的求解逻辑**：
> - **已知量**：$k^{n-1}$（上轮增殖因数）、$F^n$（当前轮全局裂变率）和 $F^{n-1}$（上轮全局裂变率）——二者均由 $\phi$、$V_i$、$\nu\Sigma_f$ 计算得到
> - **中间过程量**：$F^n$ 和 $F^{n-1}$——逐 FSR 累加 $\nu\Sigma_f \cdot \phi \cdot V$ 得到全局裂变率
> - **输出**：$k^n$——若 $k > 1$ 系统超临界（裂变率逐轮增长），$k < 1$ 次临界（逐轮衰减），$k = 1$ 临界

## 2. 一维手算例子

下面用一个极简的 1D 问题，手算 RR 的一次完整迭代。

### 2.1 问题定义

一维平板，总长度 $L = 6$ cm，分为 3 个平源区（FSR），每个宽 2 cm。1 个能群，真空边界。

| FSR | 宽度 (cm) | $\Sigma_t$ (cm⁻¹) | $\Sigma_s$ (cm⁻¹) | $\nu\Sigma_f$ (cm⁻¹) | $\chi$ |
|-----|-----------|---------------------|---------------------|------------------------|--------|
| 1 | 2 | 1.0 | 0.5 | 0.3 | 1.0 |
| 2 | 2 | 1.0 | 0.5 | 0.3 | 1.0 |
| 3 | 2 | 1.0 | 0.5 | 0.3 | 1.0 |

均匀材料，但这不重要——RR 的处理对非均匀材料完全一样。

> **本问题已知量**：FSR 几何尺寸（宽度 2 cm × 3）、各 FSR 的截面参数（$\Sigma_t$, $\Sigma_s$, $\nu\Sigma_f$, $\chi$）、真空边界条件。
> **待求未知量**：每个 FSR 的标量通量 $\phi$ 和全局有效增殖因数 $k_{eff}$。
> **求解策略**：通过幂迭代逐步逼近自洽的 $(\phi, k_{eff})$ 解——每轮迭代用上轮结果算源、输运、重构通量，循环直至收敛。

### 2.2 初始条件

设初始猜测通量 $\phi^{(0)} = 1.0$，初始 $k_{eff}^{(0)} = 1.0$。

### 2.3 步骤①：更新源

源公式（能群 $g_{out}$）：

$$Q(i, g_{out}) = \frac{1}{\Sigma_t}\left[\sum_{g_{in}} \Sigma_s(g_{in} \to g_{out}) \phi^{n-1}(g_{in}) + \frac{\chi}{k_{eff}} \sum_{g_{in}} \nu\Sigma_f(g_{in}) \phi^{n-1}(g_{in})\right]$$

其中 $Q$ 为源强度（n·cm⁻³·s⁻¹·sr⁻¹），$i$ 为 FSR 编号，$g_{in}$/$g_{out}$ 为入射/出射能群索引，$\Sigma_s(g_{in} \to g_{out})$ 为从能群 $g_{in}$ 散射到 $g_{out}$ 的宏观散射截面（cm⁻¹），$\phi^{n-1}$ 为上一轮迭代的标量通量（n·cm⁻²·s⁻¹），$\chi$ 为裂变中子谱（归一化概率分布，满足 $\sum_g \chi_g = 1$），$k_{eff}$ 为有效增殖因数，$\nu\Sigma_f$ 为中子产生截面（$\nu$ 为每次裂变平均放出的中子数，$\Sigma_f$ 为宏观裂变截面，单位 cm⁻¹），$\Sigma_t$ 为宏观总截面（cm⁻¹）。方括号内第一项为散射源贡献，第二项为裂变源贡献。

> **求解目标**：计算本轮每个 FSR 的等效源强 $Q/\Sigma_t$，供后续 MOC 衰减公式使用。
> **已知量**：$\phi^{n-1}$（上轮标量通量）、$k_{eff}^{n-1}$（上轮增殖因数）、各截面参数（$\Sigma_t$, $\Sigma_s$, $\nu\Sigma_f$, $\chi$）。
> **中间过程量**：$Q$ 和 $Q/\Sigma_t$ 本身不是最终答案，而是本轮迭代的中间输入。

1 群问题简化为：

$$Q = \frac{\Sigma_s \phi + (\chi \cdot \nu\Sigma_f \cdot \phi)/k}{\Sigma_t}$$

对所有 3 个 FSR（参数相同）：

$$Q = \frac{0.5 \times 1.0 + 1.0 \times 0.3 \times 1.0 / 1.0}{1.0} = \frac{0.5 + 0.3}{1.0} = 0.8$$

所以 $Q/\Sigma_t = 0.8$（注意代码中 `source(g)` 存的就是 $Q/\Sigma_t$）。

**对应源码**：`flat_source_domain.cpp:103-140` 的 `update_single_neutron_source()`。

### 2.4 步骤②：清零

清零 `scalar_flux_new` 和当次迭代的体积累积。

> **求解目标**：为本轮射线输运清空累积容器。`scalar_flux_new` 将用于累加每个 FSR 的 $\Delta\psi$，`volume` 用于累加射线穿行长度 $\ell$，二者必须在每轮迭代开始时归零。

**对应源码**：`flat_source_domain.cpp:79-92` 的 `batch_reset()`。

### 2.5 步骤③：射线输运

假设本次迭代采样 **3 条射线**，为简化忽略死区。

**射线 1**：从 $x=1$ 出发，向右（$\mu=+1$）

穿过 FSR 1（剩余 1 cm）→ FSR 2（2 cm）→ FSR 3（2 cm），然后从右边界出射。

**初始角通量**：$\psi^{in} = Q/\Sigma_t = 0.8$（猜测值）

MOC 衰减公式：

$$\Delta\psi = \left(\psi^{in} - \frac{Q}{\Sigma_t}\right)(1 - e^{-\Sigma_t \ell})$$

其中 $\Delta\psi = \psi^{in} - \psi^{out}$ 为角通量变化量（符号约定：$\Delta\psi > 0$ 表示角通量在穿过 FSR 时减小），$\ell$ 为射线在当前 FSR 中的穿行长度（cm），$\tau = \Sigma_t \cdot \ell$ 为光学厚度（无量纲，衡量该区域的"不透明度"），$1 - e^{-\tau}$ 为衰减因子。$\Delta\psi$ 的物理含义是：入射通量与该 FSR 平衡通量 $Q/\Sigma_t$ 的偏差，乘以衰减因子——偏差越大或光学厚度越大，通量变化越显著。

> **求解目标**：沿每条射线逐 FSR 计算角通量衰减量 $\Delta\psi$，并累加到各 FSR 的 `scalar_flux_new` 和 `volume` 中。
> **已知量**：$\psi^{in}$（射入当前 FSR 的角通量）、$Q/\Sigma_t$（步骤①已算好）、$\Sigma_t$（截面）、$\ell$（穿行长度，由几何追踪得到）。
> **中间过程量**：$\tau$（光学厚度）、$1-e^{-\tau}$（衰减因子）、$\Delta\psi$（角通量变化）、$\psi^{out}$（出射角通量，作为下一个 FSR 的 $\psi^{in}$ 传递）。

**FSR 1 段**（$\ell = 1$ cm，$\tau = 1.0 \times 1 = 1.0$）：

$$\Delta\psi_1 = (0.8 - 0.8)(1 - e^{-1.0}) = 0$$

$$\psi^{out}_1 = 0.8 - 0 = 0.8$$

> 初始猜测恰好等于源，所以 $\Delta\psi = 0$。这在第一次迭代很常见。

**FSR 2 段**（$\ell = 2$ cm，$\tau = 2.0$）：

$$\Delta\psi_2 = (0.8 - 0.8)(1 - e^{-2.0}) = 0$$

$$\psi^{out}_2 = 0.8$$

**FSR 3 段**（$\ell = 2$ cm，$\tau = 2.0$）：

$$\Delta\psi_3 = (0.8 - 0.8)(1 - e^{-2.0}) = 0$$

第一轮通量贡献全为零——这是因为初始猜测 $\psi^{in} = Q/\Sigma_t$ 恰好是方程的"平衡态"解（无限介质解）。实际三维问题中初始猜测不会如此精确。

**射线 2**：从 $x=5$ 出发，向左（$\mu=-1$）

穿过 FSR 3（剩余 1 cm）→ FSR 2（2 cm）→ FSR 1（2 cm），从左边界出射。

同理，$\Delta\psi$ 全为 0。

**射线 3**：从 $x=3$ 出发，向右（$\mu=+1$）

穿过 FSR 2（剩余 1 cm）→ FSR 3（2 cm）。

$\Delta\psi$ 依然全为 0。

### 2.6 步骤④⑤：归一化

总活跃轨迹长度 $L_{total} = 3 \times 5 = 15$ cm（3 条射线，每条活跃 5 cm——实际上这里简化了活跃长度）。

归一化因子 $= 1/L_{total} = 1/15$。

> **求解目标**：将各 FSR 累积的 $\sum\Delta\psi$ 除以所有射线的总活跃轨迹长度，得到归一化的通量贡献，消除射线数量的影响。

`scalar_flux_new` 归一化后仍为 0（因为 $\Delta\psi$ 全为 0）。

### 2.7 步骤⑥：重构标量通量

公式：

$$\phi_{i} = \frac{Q}{\Sigma_t} + \frac{\sum_r \Delta\psi_r}{\Sigma_t \cdot V_i}$$

其中 $\phi_i$ 为 FSR $i$ 的标量通量（n·cm⁻²·s⁻¹），右端第一项 $Q/\Sigma_t$ 为无限介质平衡通量（假设无泄漏时的源项贡献），第二项为输运修正：$\sum_r$ 对所有穿过 FSR $i$ 的射线 $r$ 求和，$V_i$ 为 FSR $i$ 的体积估计（cm³，由射线轨迹长度统计得到）。

> **求解目标**：重构本轮迭代的标量通量 $\phi_i$——这是每轮迭代的核心输出。
> **已知量**：$Q/\Sigma_t$（步骤①已算好）、归一化后的 $\sum_r \Delta\psi_r$ 和 $V_i$（步骤③射线输运中已累加、步骤⑤已归一化）。
> **物理含义**：$\phi_i$ 由"无限介质平衡项"加上"输运修正项"组成——如果所有射线都无衰减（$\Delta\psi = 0$），则 $\phi_i = Q/\Sigma_t$ 退化为无限介质解。

因为 $\sum_r \Delta\psi_r = 0$：

$$\phi = 0.8 + 0 = 0.8$$

**对应源码**：`flat_source_domain.cpp:187-203` 的 `set_flux_to_flux_plus_source()`。

### 2.8 步骤⑧：更新 k_eff

$$F^n = \sum_i V_i \cdot \nu\Sigma_f \cdot \phi^n$$

其中 $F^n$ 为第 $n$ 轮迭代的全局总裂变率（中子/s），$V_i$ 为 FSR $i$ 的体积估计（cm³），$\nu\Sigma_f$ 为中子产生截面（cm⁻¹），$\phi^n$ 为当前迭代的标量通量（n·cm⁻²·s⁻¹）。

> **求解目标**：根据新旧两轮通量计算全局裂变率之比，更新 $k_{eff}$。
> **已知量**：$\phi^n$（刚算好的本轮通量）、$\phi^{n-1}$（上轮通量）、$V_i$（体积估计）、$\nu\Sigma_f$（截面）。
> **中间过程量**：$F^n$（当前全局总裂变率）、$F^{n-1}$（上轮全局总裂变率）。
> **输出**：$k_{eff}^{(n)}$——更新后供下一轮源计算使用（公式中的 $1/k_{eff}$ 项）。

Naive 体积估计：$V_i = \ell_i / L_{total}$，其中 $\ell_i$ 为所有射线在 FSR $i$ 中的总穿行长度（cm），$L_{total}$ 为所有射线的总活跃轨迹长度（cm）

- FSR 1：射线 1 贡献 1 cm，射线 2 贡献 2 cm，射线 3 贡献 0 cm → $\ell_1 = 3$ cm → $V_1 = 3/15 = 0.2$
- FSR 2：射线 1 贡献 2 cm，射线 2 贡献 2 cm，射线 3 贡献 1 cm → $\ell_2 = 5$ cm → $V_2 = 5/15 = 1/3$
- FSR 3：射线 1 贡献 2 cm，射线 2 贡献 1 cm，射线 3 贡献 2 cm → $\ell_3 = 5$ cm → $V_3 = 5/15 = 1/3$

$$F^1 = 0.2 \times 0.3 \times 0.8 + \frac{1}{3} \times 0.3 \times 0.8 + \frac{1}{3} \times 0.3 \times 0.8 = 0.048 + 0.08 + 0.08 = 0.208$$

同理 $F^0$ 用 $\phi^{(0)} = 1.0$：

$$F^0 = 0.2 \times 0.3 \times 1.0 + \frac{1}{3} \times 0.3 \times 1.0 + \frac{1}{3} \times 0.3 \times 1.0 = 0.06 + 0.1 + 0.1 = 0.26$$

$$k_{eff}^{(1)} = 1.0 \times \frac{0.208}{0.26} = 0.8$$

### 2.9 第 2 次迭代（展示非平凡衰减）

新的源（$k_{eff} = 0.8$）：

$$Q = \frac{0.5 \times 0.8 + 0.3 \times 0.8 / 0.8}{1.0} = \frac{0.4 + 0.3}{1.0} = 0.7$$

$Q/\Sigma_t = 0.7$。

假设射线 1 从 $x=0.5$ 向右出发，初始角通量 $\psi^{in} = Q/\Sigma_t = 0.7$：

**FSR 1 段**（$\ell = 1.5$ cm，$\tau = 1.5$）：

$$\Delta\psi_1 = (0.7 - 0.7)(1 - e^{-1.5}) = 0$$

依然是 0！原因还是初始猜测等于源。

**关键洞察**：如果所有 FSR 的 $Q/\Sigma_t$ 相同（均匀源），且初始角通量设为 $Q/\Sigma_t$，则 $\Delta\psi$ 永远为 0——标量通量就等于 $Q/\Sigma_t$，算法退化为源迭代。

要让 $\Delta\psi \neq 0$，需要以下任一条件：
- **非均匀源**（不同 FSR 的 $Q/\Sigma_t$ 不同，射线从一个 FSR 进入另一个时角通量与源不匹配）
- **边界效应**（射线从边界进入时角通量为 0，不等于源）
- **死区效应**（射线的角通量在死区中"自然发育"为不同于源的值）

让我演示一个有**真空边界**的场景。假设射线从左边界（$\psi^{in} = 0$）进入 FSR 1：

**FSR 1 段**（$\ell = 2$ cm，$\tau = 2.0$，$Q/\Sigma_t = 0.7$）：

$$\Delta\psi_1 = (0 - 0.7)(1 - e^{-2.0}) = -0.7 \times 0.8647 = -0.6053$$

$$\psi^{out}_1 = 0 - (-0.6053) = 0.6053$$

角通量在 FSR 1 中从 0 增长到 0.6053——被源项"填充"。

**FSR 2 段**（$\ell = 2$ cm，$\tau = 2.0$）：

$$\Delta\psi_2 = (0.6053 - 0.7)(1 - e^{-2.0}) = -0.0947 \times 0.8647 = -0.0819$$

$$\psi^{out}_2 = 0.6053 + 0.0819 = 0.6872$$

继续趋近 $Q/\Sigma_t = 0.7$。

**FSR 3 段**（$\ell = 2$ cm，$\tau = 2.0$）：

$$\Delta\psi_3 = (0.6872 - 0.7)(1 - e^{-2.0}) = -0.0128 \times 0.8647 = -0.0111$$

$$\psi^{out}_3 = 0.6872 + 0.0111 = 0.6983$$

越来越接近 0.7——这就是 MOC 解析衰减的物理含义：**角通量指数趋近于 $Q/\Sigma_t$**。

### 2.10 代码对照

上面的计算过程，就是源码 `random_ray.cpp:427-476` 中 `attenuate_flux_flat_source()` 所做的事情：

```cpp
float tau = sigma_t * distance;                           // τ = Σt × ℓ
float exponential = cjosey_exponential(tau);             // ≈ 1 - exp(-τ)
float new_delta_psi = (angular_flux_[g] - srh.source(g)) * exponential;
//                    (ψ_in       -      Q/Σt          ) × (1 - e^{-τ})
angular_flux_[g] -= new_delta_psi;                       // ψ_out = ψ_in - Δψ

if (is_active) {
  srh.scalar_flux_new(g) += delta_psi_[g];               // 累加 ΣΔψ
  srh.volume() += distance;                               // 累加 ℓ（估计体积）
}
```

## 5. 幂迭代主循环

`src/random_ray/random_ray_simulation.cpp:406-496`

```cpp
void RandomRaySimulation::simulate() {
  while (simulation::current_batch < settings::n_batches) {
    if (mpi::master) {
      // ① 更新源 Q^n/Σt
      domain_->update_all_neutron_sources();

      // ② 清零本轮通量和体积
      domain_->batch_reset();

      // ③ 输运扫描（OpenMP 并行）
      #pragma omp parallel for schedule(dynamic)
      for (int i = 0; i < settings::n_particles; i++) {
        RandomRay ray(i, domain_.get());
        ray.transport_history_based_single_ray();
      }

      // ④ 合并新发现的 FSR
      domain_->finalize_discovered_source_regions();

      // ⑤ 归一化通量和体积
      domain_->normalize_scalar_flux_and_volumes(
        settings::n_particles * RandomRay::distance_active_);

      // ⑥ 重构标量通量 φ = Q/Σt + ΣΔψ/(Σt·V)
      int64_t n_hits = domain_->add_source_to_scalar_flux();

      // ⑦ 输运修正稳定化
      domain_->apply_transport_stabilization();

      // ⑧ 计算 k_eff
      domain_->compute_k_eff();

      // ⑨ 活跃批次：累积通量 + tally
      if (simulation::current_batch > settings::n_inactive) {
        domain_->accumulate_iteration_flux();
        domain_->random_ray_tally();
      }

      // ⑩ φ_old ← φ_new
      domain_->flux_swap();
    }
  }
}
```

下面逐步展开，将代码与 1D 例子中的计算严格对照。

> **每轮迭代的整体逻辑**：
> - **已知输入**：上轮通量 $\phi^{n-1}$、上轮 $k_{eff}^{n-1}$、截面数据（全迭代不变）
> - **迭代目标**：从 $\phi^{n-1}$ 出发，经 ① 源更新 → ② 清零 → ③ 射线输运 → ④⑤ 归一化 → ⑥ 通量重构 → ⑦ 稳定化 → ⑧ 更新 $k_{eff}$，得到新的 $\phi^n$ 和 $k_{eff}^n$
> - **收敛判据**：$k_{eff}$ 和 $\phi$ 不再显著变化。inactive batch 期间结果丢弃，active batch 期间累积统计

## 6. 步骤①：源更新

### 理论公式

$$Q^n(i, g_{out}) = \frac{1}{\Sigma_t}\left[\sum_{g_{in}} \Sigma_s \cdot \phi^{n-1}(g_{in}) + \frac{\chi}{k_{eff}} \sum_{g_{in}} \nu\Sigma_f \cdot \phi^{n-1}(g_{in})\right]$$

其中各变量含义同第 2.3 节的定义，上标 $n$ 表示当前迭代步。

> **已知量**：$\phi^{n-1}$（代码中 `scalar_flux_old`）、$k_{eff}^{n-1}$（`k_eff_`）、多群截面数据（`sigma_s_`, `nu_sigma_f_`, `chi_`, `sigma_t_`，全部在迭代前从输入文件读取，迭代过程中不变）。
> **中间过程量**：`scatter_source`（散射源贡献）和 `fission_source`（裂变源贡献），按出射能群 $g_{out}$ 累加。
> **输出**：`source(g_out)` = $Q/\Sigma_t$，直接存好除以 $\Sigma_t$ 后的值，供 MOC 衰减公式使用。

**注意**：代码中 `source(g)` 存储的是 $Q/\Sigma_t$，不是 $Q$ 本身。这样 MOC 衰减公式可以直接使用。

### 对应代码

`src/random_ray/flat_source_domain.cpp:103-140`

```cpp
void FlatSourceDomain::update_single_neutron_source(SourceRegionHandle& srh) {
  for (int g = 0; g < negroups_; g++) {
    srh.source(g) = 0.0;
  }

  int material = srh.material();
  if (material != MATERIAL_VOID) {
    double inverse_k_eff = 1.0 / k_eff_;
    for (int g_out = 0; g_out < negroups_; g_out++) {
      double sigma_t = sigma_t_[material * negroups_ + g_out];
      double scatter_source = 0.0;
      double fission_source = 0.0;

      for (int g_in = 0; g_in < negroups_; g_in++) {
        double scalar_flux = srh.scalar_flux_old(g_in);
        double sigma_s =
          sigma_s_[material * negroups_ * negroups_ + g_out * negroups_ + g_in];
        double nu_sigma_f = nu_sigma_f_[material * negroups_ + g_in];
        double chi = chi_[material * negroups_ + g_out];

        scatter_source += sigma_s * scalar_flux;
        fission_source += nu_sigma_f * scalar_flux * chi;
      }
      // 源已经除以 Σt
      srh.source(g_out) =
        (scatter_source + fission_source * inverse_k_eff) / sigma_t;
    }
  }

  // 固定源模式下额外加上外源
  if (settings::run_mode == RunMode::FIXED_SOURCE) {
    for (int g = 0; g < negroups_; g++) {
      srh.source(g) += srh.external_source(g);
    }
  }
}
```

**与 1D 例子的对照**：第 2.3 节中我们手算的 $Q = (\Sigma_s \phi + \chi \nu\Sigma_f \phi/k)/\Sigma_t = 0.8$，对应代码中 `scatter_source + fission_source * inverse_k_eff` 除以 `sigma_t`。

### 7.2 死区处理

`src/random_ray/random_ray.cpp:295-331`

```cpp
void RandomRay::event_advance_ray() {
  boundary() = distance_to_boundary(*this);
  double distance = boundary().distance();

  if (is_active_) {
    // 活跃区：检查是否超过活跃长度
    if (distance_travelled_ + distance >= distance_active_) {
      distance = distance_active_ - distance_travelled_;
      wgt() = 0.0;  // 终止
    }
    attenuate_flux(distance, true);   // 衰减并记录
  } else {
    // 死区
    if (distance_travelled_ + distance >= distance_inactive_) {
      // 拆成两段：死区尾 + 活跃区头
      double distance_dead = distance_inactive_ - distance_travelled_;
      attenuate_flux(distance_dead, false);      // 只衰减不记录
      double distance_alive = distance - distance_dead;
      attenuate_flux(distance_alive, true);       // 衰减并记录
      is_active_ = true;
    } else {
      attenuate_flux(distance, false);  // 全在死区内
    }
  }

  // 推进位置
  for (int j = 0; j < n_coord(); ++j)
    coord(j).r() += distance * coord(j).u();
}
```

**死区的作用**：初始角通量是个粗略的猜测（$Q/\Sigma_t$）。经过若干 mfp 的指数衰减 $e^{-\Sigma_t \ell}$ 后，猜测值的影响消失，角通量被沿途的源项重新塑造。1D 例子中为了简化忽略了死区。

### 7.3 MOC 衰减公式——方法的心脏

> **已知量**：$\psi^{in}$（`angular_flux_[g]`，初始值或上一 FSR 的出射值）、$Q/\Sigma_t$（`srh.source(g)`，步骤①已算好）、$\Sigma_t$（截面）、穿行长度 $d$（几何追踪得到）。
> **中间过程量**：$\tau = \Sigma_t \cdot d$（光学厚度）、$1-e^{-\tau}$（衰减因子）、$\Delta\psi$（角通量变化量）。
> **输出**：更新后的 $\psi^{out}$（写回 `angular_flux_[g]`，作为下一个 FSR 的入射值）；累加 $\Delta\psi$ 到 FSR 的 `scalar_flux_new`、累加 $d$ 到 `volume`。

`src/random_ray/random_ray.cpp:437-476`

```cpp

  //                    (ψ_in       -      Q/Σt          ) × (1 - e^{-τ})

  angular_flux_[g] -= new_delta_psi;  // ψ_out = ψ_in - Δψ

    srh.scalar_flux_new(g) += delta_psi_[g];  // 累加 ΣΔψ
```

**与 1D 例子的严格对照**：

| 手算步骤 | 代码行 | 说明 |
|---------|--------|------|
| $\tau = 1.0 \times 2 = 2.0$ | `tau = sigma_t * distance` | 光学厚度 |
| $1-e^{-2} = 0.8647$ | `cjosey_exponential(tau)` | 有理函数逼近 |

| 累加 -0.6053 到 FSR 1 | `srh.scalar_flux_new(g) += delta_psi_[g]` | ΣΔψ |

## 8. 步骤⑤⑥：归一化与通量重构

### 归一化

> **已知量**：`scalar_flux_new`（各 FSR 累积的 $\sum\Delta\psi$）、`volume`（各 FSR 累积的射线穿行长度 $\ell$）、总活跃轨迹长度 $L_{total} = n_{particles} \times distance_{active}$。
> **求解目标**：将累积量除以 $L_{total}$，消除射线数量的影响，得到归一化的通量贡献和体积估计。

`src/random_ray/flat_source_domain.cpp:157-185`

```cpp
void FlatSourceDomain::normalize_scalar_flux_and_volumes(
  double total_active_distance_per_iteration)  // = n_particles × distance_active_
{
  double normalization_factor = 1.0 / total_active_distance_per_iteration;

  // 通量除以总活跃轨迹长度
  for (int64_t se = 0; se < n_source_elements(); se++) {
    source_regions_.scalar_flux_new(se) *= normalization_factor;
  }

  // 更新体积估计
  for (int64_t sr = 0; sr < n_source_regions(); sr++) {
    source_regions_.volume_t(sr) += source_regions_.volume(sr);
    source_regions_.volume_naive(sr) = source_regions_.volume(sr) * normalization_factor;
    source_regions_.volume(sr) =
      source_regions_.volume_t(sr) * volume_normalization_factor;
  }
}
```

**与 1D 例子的对照**：第 2.6 节中 $L_{total} = 15$ cm，归一化因子 $= 1/15$。

### 通量重构

> **已知量**：归一化后的 $\sum\Delta\psi$（`scalar_flux_new`）、$Q/\Sigma_t$（`source`，步骤①已算好）、$\Sigma_t$（截面）、$V_i$（归一化后的体积估计）。

- **有外源**：用上一轮的通量值（避免外源在低截面区产生的巨大偏差）

## 9. 步骤⑧：k_eff 计算

> **已知量**：$\phi^n$（`scalar_flux_new`，刚重构好的本轮通量）、$\phi^{n-1}$（`scalar_flux_old`，上轮通量）、体积 $V_i$（`volume`）、$\nu\Sigma_f$（截面）。
> **中间过程量**：$F^n$（当前全局总裂变率）、$F^{n-1}$（上轮全局总裂变率）——逐 FSR 累加 $\nu\Sigma_f \cdot \phi \cdot V$ 得到。
> **输出**：$k_{eff}^n = k_{eff}^{n-1} \times F^n / F^{n-1}$——供下一轮源更新中使用。

`src/random_ray/flat_source_domain.cpp:307-373`

```cpp
void FlatSourceDomain::compute_k_eff() {
  double fission_rate_old = 0, fission_rate_new = 0;

  for (int64_t sr = 0; sr < n_source_regions(); sr++) {
    double volume = source_regions_.volume(sr);  // simulation averaged
    if (volume == 0.0 || material == MATERIAL_VOID) continue;

    double sr_fission_old = 0, sr_fission_new = 0;
    for (int g = 0; g < negroups_; g++) {
      double nu_sigma_f = nu_sigma_f_[material * negroups_ + g];
      sr_fission_old += nu_sigma_f * scalar_flux_old(sr, g);
      sr_fission_new += nu_sigma_f * scalar_flux_new(sr, g);
    }

    fission_rate_old += sr_fission_old * volume;  // F^{n-1}
    fission_rate_new += sr_fission_new * volume;  // F^n
  }

  k_eff_ = k_eff_ * (fission_rate_new / fission_rate_old);
  //       k^{n-1} × F^n / F^{n-1}

  // 同时计算 Shannon 熵
  // H = -Σ p_i log2(p_i)，其中 p_i = (νΣf × φ × V)_i / F^n
}
```

**与 1D 例子的对照**：第 2.8 节中 $k^{(1)} = 1.0 \times 0.208/0.26 = 0.8$。

```python
    azimuthal_planes.append(openmc.Plane(...))

```

## 11. 工程细节拾遗

### 11.1 动态 FSR 发现

`src/random_ray/flat_source_domain.cpp:1351-1524`

使用 mesh 细分时，FSR 不是预创建的，而是射线首次穿过时按需创建。通过线程安全的 `ParallelMap` 实现，分四种情况：已在永久 map / 在本轮临时 map / 浮点伪影（丢弃）/ 首次发现（创建）。

### 11.2 输运修正稳定化

`src/random_ray/flat_source_domain.cpp:1559-1606`

输运修正截面可能产生负的对角散射截面 $\Sigma_s(g \to g) < 0$，导致迭代不稳定。OpenMC 实现了 Gunow 2019 的对角稳定化：

$$\phi_{new} = \frac{\phi_{new} - D \cdot \phi_{old}}{1 - D}, \quad D = \rho \cdot \frac{\Sigma_s(g \to g)}{\Sigma_t(g)}$$

其中 $\phi_{new}$ 和 $\phi_{old}$ 分别为当前轮和上一轮迭代的标量通量，$D$ 为稳定化因子，$\rho$ 为用户可调的稳定化强度参数（对应代码中 `diagonal_stabilization_rho_`，默认 0.25），$\Sigma_s(g \to g)$ 为群内自散射截面（输运修正后可能为负值），$\Sigma_t(g)$ 为能群 $g$ 的宏观总截面（cm⁻¹）。

> **求解目标**：当输运修正导致对角散射截面 $\Sigma_s(g \to g) < 0$ 时，防止迭代发散——对通量做稳定化校正。
> **已知量**：$\phi_{new}$（步骤⑥刚算好）、$\phi_{old}$（上轮通量）、$\Sigma_s(g \to g)$ 和 $\Sigma_t(g)$（截面）、$\rho$（用户参数）。
> **中间过程量**：$D$（稳定化因子，$\Sigma_s(g \to g)/\Sigma_t(g)$ 越负则 $D$ 越大，校正越强）。
> **输出**：校正后的 $\phi_{new}$。

### 11.3 锁的粒度

`src/random_ray/random_ray.cpp:420-425`

每个 FSR 一把 `OpenMPMutex`，而不是逐能群原子操作。源码注释解释了原因：一次锁操作写入所有能群，比 $G$ 次 `atomic` 更高效。
