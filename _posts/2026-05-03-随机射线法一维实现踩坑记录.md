## 0. 问题设置

一维均匀平板，两侧真空边界条件，单群近似：

| 参数 | 值 |
|------|------|
| 平板厚度 $H$ | 10 cm |
| $\Sigma_t$ | 0.5 cm$^{-1}$ |
| $\Sigma_s$ | 0.4 cm$^{-1}$ |
| $\Sigma_a = \Sigma_t - \Sigma_s$ | 0.1 cm$^{-1}$ |
| $\nu\Sigma_f$ | 0.25 cm$^{-1}$（$\nu=2.5$, $\Sigma_f=0.1$）|
| $k_\infty = \nu\Sigma_f/\Sigma_a$ | 2.5 |
| 扩散理论 $k_\text{eff}$ | 1.51 |

蒙特卡罗（ch23.py）给出 $k_\text{eff} \approx 1.80$（高于扩散理论，因为输运方程无扩散近似）。

## 1. 陷阱一：路径长度的期望发散

### 错误做法

直接复用 OpenMC 三维的通量重构公式：

$$\varphi_i = \frac{Q}{\Sigma_t} + \frac{\sum \Delta\psi_i}{\Sigma_t \cdot V_i^{\text{track}}}$$

其中 $V_i^{\text{track}} = \sum \ell_i$ 是射线在 FSR $i$ 中的累积路径长度（归一化后作为体积估计）。

### 结果

$k_\text{eff}$ 收敛到 2.4–2.5，接近 $k_\infty$，泄漏被严重低估。

### 根因

在一维平板中，射线穿过 FSR 的路径长度为：

$$\ell = \frac{\Delta x}{|\mu|}$$

其中 $\mu \in [-1, 1]$ 是方向余弦，$\Delta x$ 是 FSR 宽度。路径长度的期望值：

$$E[\ell] = \Delta x \cdot E\left[\frac{1}{|\mu|}\right] = \Delta x \int_0^1 \frac{1}{\mu}\, d\mu = +\infty$$

**期望发散！** 这意味着射线路径长度的样本均值不会收敛到一个有限值——随着射线数量增加，偶尔出现的近掠射射线（$\mu \approx 0$）会贡献极大的 $\ell$，导致体积估计不稳定。

在三维中，Cauchy 公式保证了 $\sum \ell_i / L_\text{total} \to V_i / V_\text{total}$（对凸体成立）。在一维中这个保证不存在。

### 修复

不累积 $\Delta\psi$ 和 $\ell$，改为**直接累积 $\Delta\psi / \ell$**：

$$\varphi_i = \frac{Q}{\Sigma_t} + \frac{1}{\Sigma_t \cdot N_\text{rays}} \sum_j \frac{\Delta\psi_j}{\ell_j}$$

这个公式的推导：一维标量通量的定义为 $\varphi(x) = \int_{-1}^{1} \psi(x, \mu)\, d\mu$。将 MOC 衰减解沿 FSR 积分并对 $\mu$ 求和（用射线采样替代角度积分），自然得到 $\Delta\psi / \ell$ 的权重。

```python
# 错误: 累积 Δψ 和 ℓ
dpsi[fsr] += dp
vol[fsr] += ell
# ... 然后 phi = src + dpsi_norm / (Sig_t * vol_norm)

# 正确: 累积 Δψ/ℓ
dpsi_over_ell[fsr] += dp / ell
# ... 然后 phi = src + dpsi_over_ell / (Sig_t * N)
```

## 2. 陷阱二：均匀初始源导致 Δψ ≡ 0

### 错误做法

射线从平板内部随机位置出发，初始角通量取局部源值：

$$\psi_\text{init} = \frac{Q}{\Sigma_t}\bigg|_{\text{local FSR}}$$

### 结果

$k_\text{eff}$ 直接收敛到 $k_\infty = 2.5$，完全忽略泄漏。

### 根因

幂迭代的初始猜测是 $\varphi = 1$（均匀），所以 $Q/\Sigma_t$ 各 FSR 相同。射线出发时 $\psi = Q/\Sigma_t$，在第一个 FSR 内：

$$\Delta\psi = (\psi - Q/\Sigma_t)(1 - e^{-\tau}) = 0$$

由于所有 FSR 源项相同，射线穿过任何 FSR 时都有 $\psi = Q/\Sigma_t$（稳态值），$\Delta\psi$ 恒为零。通量重构退化为 $\varphi = Q/\Sigma_t$，即无限介质解——泄漏完全消失。

而且这是一个**正反馈循环**：$k$ 越大 → 源越均匀 → $\Delta\psi$ 越小 → 泄漏越小 → $k$ 更大。

### 修复

所有射线从平板边界出发，初始角通量 $\psi = 0$（真空边界条件）：

```python
if np.random.random() < 0.5:
    x = 1e-10                    # 左边界
    mu = np.random.random()      # 向右 (μ > 0)
else:
    x = thickness - 1e-10        # 右边界
    mu = -np.random.random()     # 向左 (μ < 0)
psi = 0.0                        # 真空入射
```

边界射线的 $\psi = 0 \neq Q/\Sigma_t$，立刻产生非零 $\Delta\psi$，打破对称性。射线穿过平板时角通量逐渐趋近 $Q/\Sigma_t$——这正是 MOC 衰减的物理含义：真空入射被介质源"加热"。

> **三维不需要这个技巧**：在三维中，射线从几何内部随机位置出发，初始 $\psi = Q/\Sigma_t$，经过死区（约 1 个平均自由程）后角通量已被输运过程重塑。三维中死区能正常工作是因为几何复杂度保证不同 FSR 的源值不同。一维均匀平板太简单了——所有 FSR 源值完全相同。

## 3. 陷阱三：直接用几何体积代替射线估计体积

### 错误做法

在通量重构中，用已知的几何体积 $\Delta x$ 代替射线估计的体积：

$$\varphi_i = \frac{Q}{\Sigma_t} + \frac{\Delta\psi_i^{\text{sum}}}{\Sigma_t \cdot \Delta x}$$

### 结果

数值爆炸，通量在几步迭代后达到 $10^{30}$ 量级。

### 根因

这和陷阱一本质相同。$\Delta\psi^{\text{sum}}$ 中包含了大 $\ell$（近掠射射线）的贡献，这些射线的 $\Delta\psi$ 也很大（路径长，衰减多）。除以固定的 $\Delta x$ 无法消除这种偏差。

而正确公式中除以 $\ell$（即用 $\Delta\psi/\ell$）天然抵消了路径长度的影响：

$$\frac{\Delta\psi}{\ell} = \frac{(\psi - Q/\Sigma_t)(1 - e^{-\Sigma_t \ell})}{\ell}$$

当 $\ell \to \infty$ 时，$(1-e^{-\tau}) \to 1$，所以 $\Delta\psi/\ell \to (\psi - Q/\Sigma_t)/\ell \to 0$。大 $\ell$ 射线的贡献被自然压制。

## 4. 陷阱四：k_eff 更新中的体积因子

### 错误做法

在 $k_\text{eff}$ 更新时对裂变率乘以体积：

$$k^{(n)} = k^{(n-1)} \times \frac{\sum_i \nu\Sigma_f \cdot \varphi_i^{(n)} \cdot \Delta x}{\sum_i \nu\Sigma_f \cdot \varphi_i^{(n-1)} \cdot \Delta x}$$

### 结果

在这个特定问题上不影响正确性（$\Delta x$ 全相同，比值中约掉），但如果 FSR 宽度不同则会引入偏差。

### 正确做法

一维均匀平板中 $\Delta x$ 全相同，在比值中约掉，直接用：

$$k^{(n)} = k^{(n-1)} \times \frac{\sum_i \nu\Sigma_f \cdot \varphi_i^{(n)}}{\sum_i \nu\Sigma_f \cdot \varphi_i^{(n-1)}}$$

```python
F_new = (nu * Sig_f * phi).sum()
F_old = (nu * Sig_f * phi_old).sum()
k = k * F_new / F_old
```

## 5. 总结：三维 → 一维的坑位地图

| 问题 | 三维 (OpenMC) | 一维 (本文) | 后果 |
|------|:---:|:---:|------|
| 体积估计 | 累积 $\Delta\psi$，除以 $\Sigma_t \cdot V_\text{track}$ | 累积 $\Delta\psi/\ell$，除以 $\Sigma_t \cdot N$ | 偏差 35% |
| 射线起点 | 内部随机 + 死区 | 必须从边界出发 | 收敛到 $k_\infty$ |
| 死区 | 必须（约 1 mfp） | 不需要（边界射线已天然提供正确初值） | 无影响，但不解决问题 |
| 几何体积 | 不可直接用 | 不可直接用 | 数值爆炸 |
| $k$ 更新 | 需要体积加权 | $\Delta x$ 约掉，可省略 | 非均匀网格时出错 |

**核心教训**：一维和三维的标量通量定义看起来一样（$\varphi = \int \psi\, d\Omega$），但角度积分的离散化方式完全不同。三维中 Cauchy 公式保证了射线路径长度统计量与几何体积成正比，而一维中 $\ell = \Delta x / |\mu|$ 的期望发散——这一个数学差异，让整个体积估计框架失效。

## 附录：最终正确代码（74 行）

```python
import numpy as np

def random_ray_keff(N, Sig_t, Sig_s, Sig_f, nu, thickness, Nx=20, inactive=20, active=30):
    dx = thickness / Nx
    phi = np.ones(Nx)
    k = 1.0
    k_hist = np.zeros(inactive + active)
    for cycle in range(inactive + active):
        src = (Sig_s * phi + nu * Sig_f * phi / k) / Sig_t
        dpsi_over_ell = np.zeros(Nx)
        for _ in range(N):
            if np.random.random() < 0.5:
                x = 1e-10; mu = np.random.random(); fsr = 0
            else:
                x = thickness - 1e-10; mu = -np.random.random(); fsr = Nx - 1
            if abs(mu) < 1e-10: mu = 1e-10
            psi = 0.0
            while 0 <= fsr < Nx:
                dist = ((fsr + 1) * dx - x) if mu > 0 else (x - fsr * dx)
                ell = dist / abs(mu)
                dp = (psi - src[fsr]) * (1 - np.exp(-Sig_t * ell))
                psi -= dp
                dpsi_over_ell[fsr] += dp / ell
                x += dist * np.sign(mu)
                if x <= 0 or x >= thickness: break
                fsr += int(np.sign(mu))
        phi_old = phi.copy()
        phi = src + dpsi_over_ell / (Sig_t * N)
        F_new = (nu * Sig_f * phi).sum()
        F_old = (nu * Sig_f * phi_old).sum()
        k = k * F_new / F_old if F_old > 0 else k
        k_hist[cycle] = k
    return k_hist
```
