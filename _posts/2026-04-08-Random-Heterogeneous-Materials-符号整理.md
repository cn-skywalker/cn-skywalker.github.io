# Random Heterogeneous Materials: Microstructure and Macroscopic Properties

**作者**：Salvatore Torquato

**出版**：Springer, 2002

本书系统地建立了描述随机异质材料微观结构-宏观性质关联的统计力学框架。本篇笔记整理书中涉及的符号定义、核心概念及关键公式。

# 主要符号表

## 状态与相

| 符号 | 含义 | 备注 |
||  |  |
|------|------|------|
| $\mathcal{V}$ | 系统总体积 | $d$ 维空间中的区域 |
| $d$ | 空间维度 | 通常 $d=1,2,3$ |
| $N$ | 相的数量 | |
| $\phi_i$ | 第 $i$ 相的体积分数 | $\sum_{i=1}^{N} \phi_i = 1$ |
| $\phi$ | 某一指定相的体积分数 | 在两相系统中常用 |
| $\phi_1$ / $\phi_2$ | 相 1 / 相 2 的体积分数 | $\phi_1 + \phi_2 = 1$ |

## 相关函数

| 符号 | 含义 | 定义/公式 | 备注 |
|------|------|-----------|------|
| $S_1(\mathbf{x})$ | 一阶指示函数 | $S_1(\mathbf{x}) = \begin{cases} 1, & \mathbf{x} \in \text{相 1} \\ 0, & \text{其他} \end{cases}$ | |
| $S_2(\mathbf{x}_1, \mathbf{x}_2)$ | 二阶指示函数 | $S_2 = S_1(\mathbf{x}_1) S_1(\mathbf{x}_2)$ | |
| $S_n(\mathbf{x}_1, \ldots, \mathbf{x}_n)$ | $n$ 阶指示函数 | | |
| $S_2(r)$ | 两点相关函数 | $S_2(r) = \langle S_1(\mathbf{x}) S_1(\mathbf{x}+\mathbf{r}) \rangle$ | 平稳各向同性介质 |
