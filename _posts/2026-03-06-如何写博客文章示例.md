这是一篇示例博客文章，展示如何在你的 Jekyll 站点上写博客。

## 文章结构

每篇博客文章需要包含两部分：

### 1. Front Matter（前置元数据）

在文件开头用 `---` 包裹的 YAML 配置：

- **title**: 文章标题
- **date**: 发布日期（格式：YYYY-MM-DD）
- **permalink**: 文章 URL（可选，不写会自动生成）
- **tags**: 标签列表（用于分类和搜索）
- **categories**: 分类列表（可选）

### 2. 正文内容

使用 Markdown 语法编写正文内容。

```markdown

```

## 注意事项

1. **文件名格式**：`YYYY-MM-DD-文章标题.md`（使用英文或拼音，避免中文）
2. **日期**：使用实际发布日期
3. **标签和分类**：有助于读者找到相关文章
4. **保存位置**：所有博客文章放在 `_posts/` 目录下

## 发布流程

1. 在 `_posts/` 目录创建新的 `.md` 文件
2. 填写 front matter 和正文内容
3. 提交到 Git 仓库
4. GitHub Pages 会自动构建并发布
